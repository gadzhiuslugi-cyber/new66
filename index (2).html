⚡ Способ 1 — Google Apps Script прокси (рекомендуется)
Принцип: Apps Script запускается на серверах Google, скачивает файл с Яндекс Диска и отдаёт его дашборду — никакого CORS. Настройка один раз, работает навсегда.
Вставьте URL после развёртывания скрипта ниже
📋 НАСТРОЙКА (5 минут, один раз):
1. Откройте script.google.com → «Новый проект»
2. Вставьте код ниже → Сохранить (Ctrl+S)
3. «Развернуть» → «Новое развёртывание» → Тип: Веб-приложение
4. Доступ: Все (анонимные) → Развернуть → Скопируйте URL
5. Вставьте URL выше → нажмите «Загрузить через прокси»
